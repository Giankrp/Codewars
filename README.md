# Papas - Small utilities (Dart & Go)

Proyecto con ejemplos y utilidades en Dart y Go. Archivos:

- `hola.dart` - suma elementos de dos listas. Función principal: `arrayPlusArray`.
- `hola2.dart` - determina el trimestre del mes. Función principal: `quarter`.
- `likes.dart` - formatea mensajes de "likes". Función principal: `likes`.
- `nega.dart` - devuelve el opuesto de un número. Función principal: `opposite`.
- `pludig.dart` - calcula la raíz digital. Función principal: `digitalRoot`.
- `papas.dart` - invierte las palabras en una cadena. Función principal: `reverseWords`.
- `sumAraays.dart` - suma elementos excluyendo menor y mayor. Función principal: `sumArray`.
- `papas.go` - utilidades en Go: ordenado, morse, camelCase, etc.

Cómo ejecutar (ejemplos):
- Dart:
  - Ejecutar un archivo: `dart run <file>.dart`
  - Ejemplo: `dart run pludig.dart`
- Go:
  - Ejecutar: `go run papas.go`

Notas:
- Se han añadido comentarios explicativos en cada archivo para facilitar su lectura.
- El README resume las funciones principales y cómo ejecutar los ejemplos.
